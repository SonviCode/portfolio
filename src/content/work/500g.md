#### 500 cooking app

<br/>
<u>Stack </u> :

- Backend : TheMealDB API
- Frontend : React, Tailwind, Swiper, Axios

<br/>

500g cooking app est une application de cuisine type 750g ou marmiton... Son but est de lister différentes recettes et ingrédients avec une belle interface depuis une API publique. Les données sont en anglais.

<br/>

<u>Liste des features </u> :

- Site entièrement responsive
- Liste de toutes les recettes
- Recette random
- Rechercher une recette par son nom
- Visualiser une recette, voir toutes ses caractéristiques
- Voir tout les ingrédients
- Dépendances entre les ingrédients et les recettes, liste de toutes les recettes selon les ingrédients et inverse
- Filtre des recettes selon origine ou catégorie
- Mise en place d'une pagination si trop de données
- Slider avec Swiper
