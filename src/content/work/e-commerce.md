#### E-commerce Montagne Addicte

<br/>
Stack :

- Backend : Node.JS, Express, MongoDB, JWT, Stripe
- Frontend : React, Next.JS, Typescript, Redux Toolkit, Tailwind, Stripe

<br/>

Montagne Addicte est un E-commerce proposant des produits outdoors. Il s'agit de ma première grosse application FullStack.

<br/>

<u>Liste des features </u> :

- Site entièrement responsive
- Authentification avec JWT
- Ajouter un produit en favoris
- Ajouter un produit au panier
- Mise en place d'un historique des produits
- Filtrer par catégorie, types, prix...
- Ajouter une adresse de livraison
- Payer avec Stripe : création d'un compte Stripe avec dashboard de visualisation des commandes, possibilité de payer avec une carte bancaire, traitement des erreurs de carte
- Création de commandes et de factures, visualisation de l'historique des commandes
- Slider avec Swiper
